# Advent of Code 2016, day 2

Day 2: [Bathroom Security](https://adventofcode.com/2016/day/2)

Solution in Vim.


## Usage

```bash
vim -S keypad.vim -R <input-file>
```

To (re-)solve the current buffer:

```vim
:KeypadSolve
```


## Run

```bash
vim -S keypad.vim -R example.txt
vim -S keypad.vim -R input.txt
```
